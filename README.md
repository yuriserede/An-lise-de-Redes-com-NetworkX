O que esse código faz:
Cria uma rede com 10 usuários.
Gera conexões aleatórias entre eles.
Calcula a métrica Betweenness Centrality, que identifica quais nós atuam como “pontes” na rede.
Mostra qual usuário possui maior influência.
Exibe o grafo visualmente, destacando o usuário mais influente em vermelho.
